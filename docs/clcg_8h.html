◆ clcg_axfunc_ptr

typedef void(* clcg_axfunc_ptr) (void *instance, const clcg_complex *x, clcg_complex *prod_Ax, const int x_size, matrix_layout_e layout, complex_conjugate_e conjugate)

Callback interface for calculating the complex product of a N*N matrix 'A' multiplied by a complex vertical vector 'x'.

参数
instanceThe user data sent for the clcg_solver() functions by the client.
xMultiplier of the Ax product.
AxProduct of A multiplied by x.
x_sizeSize of x and column/row numbers of A.
layoutWhether to use the transpose of A for calculation.
conjugateWhether to use the conjugate of A for calculation.

◆ clcg_progress_ptr

typedef int(* clcg_progress_ptr) (void *instance, const clcg_complex *m, const lcg_float converge, const clcg_para *param, const int n_size, const int k)

Callback interface for monitoring the progress and terminate the iteration if necessary.

参数
instanceThe user data sent for the clcg_solver() functions by the client.
mThe current solutions.
convergeThe current value evaluating the iteration progress.
n_sizeThe size of the variables
kThe iteration count.
返回值
intZero to continue the optimization process. Returning a non-zero value will terminate the optimization process.

◆ matrix_product()

void matrix_product ( clcg_complex **  A,
const clcg_complex *  x,
clcg_complex *  Ax,
int  m_size,
int  n_size,
matrix_layout_e  layout = Normal,
complex_conjugate_e  conjugate = NonConjugate 
)

calculate product of a complex matrix and a complex vector

the product of two complex vectors are defined as <a, b> = \sum{\bar{a_i}\cdot\b_i}. Different configurations: layout=Normal,conjugate=false -> A layout=Transpose,conjugate=false -> A^T layout=Normal,conjugate=true -> \bar{A} layout=Transpose,conjugate=true -> A^H

参数
Acomplex matrix A
[in]xcomplex vector x
Axproduct of Ax
[in]m_sizerow size of A
[in]n_sizecolumn size of A
[in]layoutlayout of A used for multiplication. Must be Normal or Transpose
[in]conjugatewhether to use the complex conjugate of A for calculation
