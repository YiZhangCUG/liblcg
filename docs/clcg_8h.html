◆ clcg_axfunc_ptr

typedef void(* clcg_axfunc_ptr) (void *instance, const lcg_complex *x, lcg_complex *prod_Ax, const int x_size, matrix_layout_e layout, complex_conjugate_e conjugate)

Callback interface for calculating the complex product of a N*N matrix 'A' multiplied by a complex vertical vector 'x'.

参数
instanceThe user data sent for the clcg_solver() functions by the client.
xMultiplier of the Ax product.
AxProduct of A multiplied by x.
x_sizeSize of x and column/row numbers of A.
layoutWhether to use the transpose of A for calculation.
conjugateWhether to use the conjugate of A for calculation.

◆ clcg_progress_ptr

typedef int(* clcg_progress_ptr) (void *instance, const lcg_complex *m, const lcg_float converge, const clcg_para *param, const int n_size, const int k)

Callback interface for monitoring the progress and terminate the iteration if necessary.

参数
instanceThe user data sent for the clcg_solver() functions by the client.
mThe current solutions.
convergeThe current value evaluating the iteration progress.
n_sizeThe size of the variables
kThe iteration count.
返回值
intZero to continue the optimization process. Returning a non-zero value will terminate the optimization process.
